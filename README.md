# UTS Personal Finance API

Tugas UTS Web Advanced Development - Personal Finance API.
Disusun oleh: Anandra Syarifudin

## Langkah Menjalankan:
1. Pastikan sudah menjalankan `npm install`.
2. Jalankan `npx prisma migrate dev --name init` untuk setup database.
3. (Opsional) Jalankan `npm run seed` untuk mengisi database dengan data awal.
4. Jalankan `node index.js` untuk menghidupkan server.
